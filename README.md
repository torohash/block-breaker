# BlockBreaker

Unity で作る block breaker プロジェクトです。

## Directories

- 新規アセットはこの README の `Assets` 直下ポリシーに従って配置する
- `Assets/Scenes` - 本番用のシーンを置く
- `Assets/Scripts/Core` - ゲーム全体の進行や状態管理のスクリプトを置く
- `Assets/Scripts/Gameplay` - パドル、ボール、ブロックなど遊びの本体スクリプトを置く
- `Assets/Scripts/UI` - HUD やメニューなど UI 関連スクリプトを置く
- `Assets/Prefabs` - 再利用する GameObject を置く
- `Assets/Sprites` - スプライト画像を置く
- `Assets/Audio` - BGM と効果音を置く
- `Assets/Materials` - マテリアルを置く
- `Assets/Data` - ScriptableObject や設定データを置く
- `Assets/Sandbox` - 実験用のシーンや一時的な検証用アセットを置く
